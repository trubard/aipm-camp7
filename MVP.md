# 오늘의 일상 연구팀 · AI 팀장 — MVP 설계

고립 청년이 자기 주변을 사진으로 찍어 보내면, AI 팀장이 5분 정도의 정리 과업을
제시하고 완료 경험을 함께 쌓아 실행 리듬을 회복하도록 돕는 챗봇.

## 1. INPUT (입력)

- 형태: 사진 1장이 기본 입력, 텍스트 설명은 선택.
- 입력 방식: 사진 촬영/업로드 버튼 + 한 줄 설명 입력칸.
- 빠른 응답: 과업 카드의 "했어요 / 다른 거" 버튼.
- 빈 입력: 사진도 글도 없으면 전송되지 않음.
- 언어: 한국어.

## 2. PROCESS (처리)

- 모델: Claude Haiku 4.5 (이미지 입력 지원).
- 시스템 프롬프트: 6블록 구조 (역할 / 과제 / 출력형식 / 규칙 / 예시 / 예외).
- 동작: 사진과 감정을 읽고, 상황을 분류한 뒤, 5분짜리 과업과 응답을 생성.
- 출력 형식: JSON { reply, task, intent } 로 고정.
- 기억: 사용자별 완료 기록을 저장해 다음 대화에 이어 붙임.

## 3. OUTPUT (출력)

- 형태: 말풍선(응답) + 과업 카드(과업이 있을 때).
- 의도(intent) 4종: 과업 제시 / 응원 / 완료 축하 / 위기 대응.
- 위기 대응 시: 과업 대신 지지 메시지와 상담 전화 안내.
- 길이: 2~4문장, 화면이 깨지지 않도록 폭 제한.
- 진행 표시: 완료한 일의 누적 수.
- 실패 시: 오류 안내 메시지를 띄우고, 보낸 입력은 되돌림.

## 범위 밖 (다음 단계)

- 출근 인사 + 컨디션 선택지
- 결과 사진 인증과 칭찬
- 경험치 / 레벨 / 가상 재화
- 입력 길이 제한, 빈 입력 안내 등 입력 가드
